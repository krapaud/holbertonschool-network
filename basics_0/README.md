# Networking Basics #0

Ce dossier contient des exercices sur les concepts fondamentaux des réseaux informatiques.

## Objectifs d'apprentissage

- Comprendre le modèle OSI
- Identifier les différents types de réseaux
- Distinguer les adresses MAC et IP
- Comprendre les protocoles UDP et TCP
- Connaître les ports TCP et UDP courants
- Tester la connectivité réseau

## Fichiers

### 0-OSI_model

Réponses sur le modèle OSI (Open Systems Interconnection).

### 1-types_of_network

Questions sur les différents types de réseaux (LAN, WAN, Internet).

### 2-MAC_and_IP_address

Réponses concernant les adresses MAC et IP.

### 3-UDP_and_TCP

Différences entre les protocoles UDP et TCP.

### 4-TCP_and_UDP_ports

Script Bash qui affiche les ports TCP/UDP en écoute sur la machine locale.

```bash
./4-TCP_and_UDP_ports
```

### 5-is_the_host_on_the_network

Script Bash qui ping une adresse IP passée en argument 5 fois.

```bash
./5-is_the_host_on_the_network 8.8.8.8
```

## Utilisation

Les scripts Bash doivent être rendus exécutables :

```bash
chmod +x <nom_du_script>
```
